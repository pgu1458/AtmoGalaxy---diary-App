# AtmoGalaxy

지도 위에 날씨랑 여행 기록을 남기는 PWA 앱입니다.

🔗 [배포 링크](https://rainbow-lollipop-859583.netlify.app)

---

## 만든 이유

여행 갔을 때 그날 날씨랑 기분을 같이 기록하고 싶었는데
마땅한 앱이 없어서 직접 만들었고 sns를 보다 릴스에서 
어플을 만드는 과정을 보며 관심이 갔습니다.
또 버킷리스트들과 재밌는 기능들을 추가해보고 싶었습니다.
지도 위에 핀을 꽂듯이 기록하면 나중에 봤을 때
어디서 어떤 날씨였는지 한눈에 보이는 게 목표였습니다.

---

## 주요 기능

- 실시간 날씨 히트맵**
한국·일본 주요 도시 날씨를 색깔로 표현합니다.
맑으면 노란색, 비면 파란색, 뇌우면 보라색 이런 식으로
줌 레벨에 따라 히트맵이랑 마커가 자동으로 전환됩니다.

- 여행 일기 기록**
지도 아무데나 탭하면 그 위치 날씨가 자동으로 저장되고
사진이랑 메모도 같이 남길 수 있습니다.
일기 쓰듯이 그날 감성을 기록하는 용도입니다.

- 3D 도시 뷰**
지도를 확대하면 건물이 3D로 솟아오릅니다.
내가 기록한 장소를 입체적으로 볼 수 있습니다.

- AI 버킷리스트**
Google Gemini API로 나이대에 맞는 목표를 추천해줍니다.
여행, 자격증, 건강, 재정 등 카테고리별로 추천받고
완료하면 지도 핀이 초록색으로 바뀝니다.
API 할당량 초과 시 자체 DB로 자동 전환됩니다.

- 랜덤 여행 추첨**
어디 갈지 못 정하겠을 때 사용합니다.
한국/일본/전체 범위에서 랜덤으로 추첨하고
다트 날아가는 애니메이션이랑 Wikipedia 사진도 자동으로 불러옵니다.

- 여행 통계**
쌓인 기록을 분석해서 평균 기온, 자주 만난 날씨,
요일별 기록 수 같은 통계를 보여줍니다.
날씨 패턴에 따라 날씨 요정 캐릭터 카드도 나옵니다.

## 기술 스택

| 분류 | 사용 기술 |
|------|----------|
| Frontend | React 18, TypeScript, Vite |
| 지도 | Mapbox GL JS |
| 애니메이션 | Framer Motion |
| 차트 | Recharts |
| 날씨 | OpenWeather API |
| AI | Google Gemini 1.5 Flash |
| 이미지 | Wikipedia REST API |
| 스토리지 | localStorage |
| PWA | vite-plugin-pwa + Workbox |
| 배포 | Netlify |

## 개발하면서 신경 쓴 것들

- iPhone Safari에서 안전 영역(Safe Area) 대응
- 검색창 포커스 시 iOS 자동 줌인 방지
- 드래그로 높이 조절 가능한 바텀 시트 패널
- PWA 캐시 설정으로 오프라인에서도 기본 동작
- Gemini API 할당량 초과 시 폴백 DB 자동 전환
- 네이티브 공유 기능 (카카오톡, 메일 등)
